# Unit-Testing-BankAccount
Kelas BankAccount dipilih sebagai objek pengujian karena memiliki beberapa metode yang mengandung logika bisnis, seperti validasi input, perubahan nilai saldo, serta pengecekan kondisi tertentu.  Dengan menggunakan unit testing, setiap metode dapat diuji secara terpisah untuk memastikan bahwa program berjalan sesuai dengan yang diharapkan
